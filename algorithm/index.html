# 项目背景

作为前端开发，本身业务需求对算法的要求不高，也导致大家对算法不够重视，而我在偶尔做 LeetCode 的时候也很难找到通俗易懂的 Js 题解。

于是就有了《算法|进阶》部分，尽自己的努力整理 LeetCode 题解 Js 版本，如果该项目能对大家起到积极的作用，就算成功了，欢迎大家留言交流，接受批评和指正～
